### Instructions

1. Select the number of wins from the drop-down list in "A race to...". (default is 3). <br/>

2. Choose you option â€” Rock, Paper, or Scissor. <br/> (Your choice and the computer's choice will appear above the result box.)

3. Once the game is finished, the winner will be announced below your screen. You can then choose to start a new game or play a different round. <br/>

(You can also click the New Game or choose a new winning score even though the current game is not done. Doing so will reset the current game.) <br/>
